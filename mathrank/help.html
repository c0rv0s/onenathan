Problem Sets

Problem sets run on a timer. Each set is two minutes and students receive ten points for each correct answer. At the end of the timer period their final score is displayed on the screen.


Multiplication tables run in sequential order. After selecting the starting set each multiplication table runs to 12 and then resets with the next set (eg. after 4 x 12 the next question is 5 x 2, 5 x 3, etc.).


All other math problem sets are generated in real time based on our algorithms to provide fresh questions that will continue to challenge students problem solving skills. The division set asks students to round their answers down to the nearest whole number for problems that have a remainder (eg. 10 / 3 = 3, 100 / 99 = 1, 100 / 51 = 1). Future versions of MathRank may include questions that ask the student to enter the remainder to get the question correct.
